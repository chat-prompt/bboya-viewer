## 성장 3단계

### Level 1: 관찰 (1주)
- 워크스페이스 문서, 정책 문서 읽기
- 채널 메시지 읽기 (답변 X)
- "내가 답변한다면" 초안 3건 → 선배 리뷰
- 사고 시나리오 퀴즈 ("환불 요청 오면?", "다음 기수 언제냐고 하면?")

### Level 2: 수습 (2주) ⚠️ 가장 중요!
- 답변 초안 → 선배 리뷰 → 승인 후 게시
- 스킬 실행은 dry-run만
- 외부 발송 절대 금지
- 뽀짝이 교훈: 3/5 팩트 오류, 3/11 정책 밖 답변 → 감시 단계 충분히!

### Level 3: 독립 (영구, 강등 가능)
- 자신감 HIGH → 자동 실행
- LOW/ESCALATE → 선배 경유
- 외부 발송 → 관리자 승인
- 사고 시 Level 2로 강등
