# 반복 질문 패턴 (FAQ)

> 덕후방에서 반복 등장하는 질문. 콘텐츠 소재 + 봇 답변 개선 자료.
> ★ = 등장 빈도, 📚 = 현존 자원, 🔲 = 콘텐츠 갭

---

## 1. 카톡봇 만드는 법 ★★★★★

**대표 질문**: "카톡봇 어떻게 만들어요?" "카톡방에 봇 넣으려면?"

**질문 변형**:
- "카카오채널에 봇 만드는법 알려줘" (클로찡 3/17)
- "카톡봇 진짜 해보고 싶은데 하는 법 1도 몰라" (하이호 3/17)
- "오픈클로로 카톡방 봇 만드는 법 문서가 있을까요" (토옵이 3/19)
- "카톡 응답을 어떤 방식으로 하고 계신가요?" (곽은철 3/24)
- "단톡방에도 봇 초대가 가능할까요?" (루비 3/24)

**현재 답변 패턴**:
- 닿: "서브폰, 별도 번호 필요" + 물결님 사례글 링크
- 클로찡: "카톡CLI 검색하세요"
- 소파: lesson-11 링크 공유
- 바르톨로메오: 메신저R/알림감지/GUI 등 기술 방식 소개

📚 lesson-11, 물결님 사례글
🔲 **end-to-end 원데이 가이드** (서브폰 구매 → 계정 → 연결 → 첫 응답)
