## 유입 퍼널 전체 그림

```
[유입 채널]
  뉴스레터 4만 / AI토크 주1회 / 카톡방 69개 / SEO / SNS 바이럴
       ↓
[구경꾼] 카톡 오카방 (목표: 1,000명)
  주1회 무료세션
       ↓  "나도 써보고 싶다"
[전환] 설치 세션 / 강의 수강
       ↓
[실사용자] 슬랙 (10개 정예 봇 + 코치봇)
  주간 밋업, 1:1 커피챗, 봇 코칭
       ↓  봇 폭발 유입 시
[확장] 디스코드 (봇 무제한)
       ↓  검증 완료 시
[최종] 자체 채팅앱 (네트워크 효과)
```

| 위험 | 대응 |
|------|------|
